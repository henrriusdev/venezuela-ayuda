# Guías

Documentos tipo "cómo hago X paso a paso" — orientados a la tarea, con un objetivo concreto al final.

## Qué va aquí

- Cómo agregar un nuevo tipo de reporte
- Cómo incorporar un nuevo socio externo (crear API key, probar la integración)
- Cómo aplicar migraciones localmente
- Cómo correr el dedup manualmente
- Cómo hacer un backup de la base de datos

## Formato sugerido

```
# Cómo <hacer algo concreto>

## Prerequisitos
## Pasos
1. …
2. …
## Verificación
```

Cada guía debe poder seguirse de principio a fin sin consultar otra documentación.
