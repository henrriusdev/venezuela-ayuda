# Referencia

Datos de referencia secos — sin explicación narrativa. El lector ya sabe qué busca y viene a consultar un dato puntual.

## Qué va aquí

- Catálogo de enums (`category`, `urgency`, `severity`, `status`) y sus valores válidos
- Esquema de tablas y vistas `public_*` (campos, tipos, restricciones)
- Bounding box geográfico de Venezuela usado para validar coordenadas
- Formato y campos del spec OpenAPI (`/api/openapi`)
- Lista de namespaces de i18n y convenciones de claves de traducción

## Formato sugerido

Tablas, listas y bloques de código. Sin prosa innecesaria — es material de consulta rápida.
