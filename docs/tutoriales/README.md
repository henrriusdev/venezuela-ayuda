# Tutoriales

Aprendizaje guiado — orientados a que alguien nuevo entienda el proyecto haciendo. A diferencia de las guías, el objetivo es el aprendizaje en sí, no solo completar una tarea.

## Qué va aquí

- Recorrido por el proyecto para un colaborador nuevo (setup local → primer cambio → PR)
- Cómo funciona el flujo completo de un reporte: desde el form hasta la vista pública
- Introducción al modelo de auditoría y trazabilidad

## Formato sugerido

Narrativo y progresivo — cada paso construye sobre el anterior. Incluir contexto de "¿por qué hacemos esto?" además del "¿cómo?".
